API

              
外部連携用 API 一覧

            

          

          

            

              

                GET
                /api/bootstrap
              

              
現在の状態、スケジュール、進行ログをまとめて取得します。

            

            

              

                GET
                /api/schedule
              

              
スケジュール配列のみ取得します。

            

            

              

                PUT
                /api/schedule
              

              
{ "schedule": [...] } を送って全スケジュールを更新します。

            

            

              

                GET
                /api/saved-schedules
              

              
保存済みスケジュール一覧を取得します。

            

            

              

                POST
                /api/saved-schedules
              

              
{ "name": "...", "schedule": [...] } を送って名前付き保存します。

            

            

              

                POST
                /api/saved-schedules/load
              

              
{ "name": "..." } を送って保存済みスケジュールを読み込みます。

            

            

              

                PUT
                /api/timers
              

              
{ "timers": [{ "id": 1, "initialValue": 180 }] } 形式で初期値を更新します。

            

            

              

                POST
                /api/timer/:id/:action
              

              
タイマーの start / pause / reset を実行します。

            

            

              

                PUT
                /api/dashboard-config
              

              
ダッシュボード表示設定を保存します。

            

            

              

                POST
                /api/offset
              

              
{ "value": 30 } のように秒単位でオフセットを加算します。

            

            

              

                POST
                /api/schedule/preview/previous
              

              
イベントカードの「戻す」に相当するプレビュー候補を返します。

            

            

              

                POST
                /api/schedule/preview/next
              

              
イベントカードの「次へ」に相当するプレビュー候補を返します。

            

            

              

                POST
                /api/schedule/shift/previous
              

              
イベントカードの「戻す（強制）」に相当する進行移動を実行します。

            

            

              

                POST
                /api/schedule/shift/next
              

              
イベントカードの「次へ（強制）」に相当する進行移動を実行します。

            

            

              

                POST
                /api/resync
              

              
{ "id": "evt_003" } を送って指定イベントへ同期します。

            

            

              

                POST
                /api/schedule/shift
              

              
{ "action": "previous" } / next / hold を送って進行位置を移動します。

            

            

              

                DELETE
                /api/progress-log
              

              
進行ログを全消去します。

            

            

              

                POST
                /api/reset
              

              
状態、スケジュール、進行ログを初期値へ戻します。
